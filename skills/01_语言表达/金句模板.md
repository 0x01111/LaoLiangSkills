# 金句模板

## 技能概述
金句是老梁输出内容的"弹药"，高度凝练、易于传播，往往一句话胜过千言万语。

---

## 一、金句结构类型

### 类型一：对比型
```
【A 有多强/大/好】，【B 就有多弱/小/差】
```
- 示例：*"赢得有多轻松，输得就有多惨烈。"*

### 类型二：反转型
```
【表面上看是 A】，【实际上是 B】
```
- 示例：*"表面上是在救人，实际上是在救自己。"*

### 类型三：因果型
```
【不是因为 A 才 B】，【而是因为 C 才 B】
```
- 示例：*"他不是因为有钱才自信，而是因为自信才有钱。"*

### 类型四：定义颠覆型
```
【什么叫真正的 X】？【反常识的定义】
```
- 示例：*"什么叫真正的强者？是输得起，不是赢得多。"*

### 类型五：历史映现型
```
【历史上 XXX 说过/做过】，【今天看来依然如此】
```
- 示例：*"古人说：知己知彼，百战不殆。放到今天的商场，一个字没变。"*

## 三、金句打磨四步法

1. **提炼核心观点** → 这句话要传达什么？
2. **压缩语言长度** → 能用10个字说清楚，不用20个字
3. **增加对比/反转** → 制造认知落差，让人印象深刻
4. **测试传播性** → 这句话别人能不能轻松复述？

---

## 四、金句使用时机

- 开场：定下基调，抓住注意力
- 转折点：强化逻辑转变
- **结尾**：最佳位置，留下最深印象
- 高潮处：配合情绪渲染，引爆共鸣
