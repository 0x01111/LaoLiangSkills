# 老梁（梁宏达）风格评论技能包

## 关于老梁

梁宏达，人称"老梁"，中国著名电视评论人、脱口秀主持人。其风格以深厚的历史积累、犀利的社会洞察力、独特的语言表达著称，融合了北方评书传统与新闻评论的精髓。

## 风格特征

- **"话说……"**：经典开场白，用故事引入话题
- **古今贯通**：用历史典故映照现实问题
- **接地气**：语言通俗易懂，市井与学识兼备
- **反常识**：挑战大众固有认知，提供新视角
- **数据驱动**：论证时引用具体数字和案例

## 技能模块

本技能包包含七个模块，每个模块下有多个技能文件：

| 模块 | 路径 | 技能 |
|------|------|------|
| 01 语言表达 | 口才技巧、幽默风格、金句模板 |
| 02 历史解读 | 历史分析框架、朝代知识库、历史人物解读 |
| 03 社会评论 | 时事分析方法、社会现象解读、民生话题切入 |
| 04 体育评论 | 赛事解说技巧、运动员分析、体育精神解读 |
| 05 故事叙述 | 叙事结构、情节设计、代入感技巧 |
| 06 逻辑思维 | 辩证分析法、反驳技巧、论证结构 |
| 07 娱乐评论 | 影视点评方法、明星解读角度、流行文化分析 |

## 使用方式

在 Claude Code 中通过 `/lm` 引用对应技能文件，即可获得老梁风格的写作指导：

```
/lm 你是一位资深评论人，请参考 03_社会评论/时事分析方法 中的框架，
用老梁的风格写一篇关于[话题]的评论
```

每个技能文件都包含该领域的具体技巧模板、金句公式和注意事项，可根据需要灵活调用。
